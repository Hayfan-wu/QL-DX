# QL-DX - 中国电信话费自动化

基于 API 直调的中国电信网上营业厅自动化脚本，支持**青龙面板定时运行** + **QQ 机器人交互控制** + **产物自动记录查询**。

> **v3.0**：API 直调模式，无需 Chromium。单文件 `dx_auto.py` 入口，与 QL-WPS 模式一致。

## 功能

| 功能 | 说明 |
|------|------|
| 服务密码登录 | Token 缓存，24 小时内免重复登录 |
| 每日签到 | 签到翻牌领金豆 |
| 连签兑换 | 7天/15天/28天连签奖励自动兑换 |
| 首页任务 | 领取奖励/看视频/聚合任务自动完成 |
| 宠物乐园 | 喂食/升级/兑换话费 |
| 金豆查询 | 余额及过期提醒 |
| 产物记录 | 每次执行自动记录产物，可随时查询 |
| 瑞数绕过 | 内置 execjs 反爬处理 |

## 依赖

```
pip install httpx PyExecJS pycryptodome requests
```

## 青龙面板

定时任务命令：`task dx_auto.py`

## QQ 机器人

配合 [QL-Bot](https://github.com/Hayfan-wu/QL-Bot) 使用，自动扫描 `/opt/QL-DX/bot_plugins/` 加载插件。

| 命令 | 功能 |
|------|------|
| `电信菜单` | 帮助菜单 |
| `电信登录` | 多轮引导设置账号密码，自动提交青龙 |
| `电信状态` | 查看配置状态 |
| `电信查询` | 查询历史产物 |
| `电信签到` | 手动签到 |
| `电信执行` | 执行全部任务 |
| `电信配置` | 查看环境变量 |
| `电信开启/关闭 签到/活动/秒杀` | 功能开关 |

## 项目结构

```
QL-DX/
├── dx_auto.py          # 唯一入口（青龙定时任务用这个）
├── rs_core.js           # 瑞数反爬 JS 环境
├── .env                 # 环境变量（git不追踪）
├── .env.example         # 环境变量模板
├── result.json          # 产物记录（自动生成）
├── bot_plugins/         # QQ机器人插件
│   └── dx_plugin.py     # 交互逻辑
├── config.py            # 旧配置文件（已合并到 dx_auto.py）
├── telecom_api.py       # 旧模块文件（已合并到 dx_auto.py）
└── main.py              # 旧入口文件（已合并到 dx_auto.py）
```

## 注意事项

- 需要 Node.js 运行时（execjs 依赖）
- `DX_ACCOUNT` 格式为 `手机号#密码`
- 产物记录保存在 `result.json`，通过 `电信查询` 查看
